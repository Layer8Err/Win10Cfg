# Win10Cfg
Config Windows 10 for general dev work

## Getting Started

1. [Download files as a zipfile](https://github.com/Layer8Err/Win10Cfg/archive/master.zip)
2. Unzip files
3. Run SetupAll.ps1 as Admin (twice)
